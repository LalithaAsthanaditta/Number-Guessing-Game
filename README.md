# About

This game challenges players to guess a randomly generated number within a set range and limited attempts. It features multiple difficulty levels, instant feedback, and a celebratory confetti animation on winning!


# Features

- Choose between Easy, Medium, and Hard difficulty levels
- Limited attempts for added challenge
- Real-time feedback: too high, too low, or correct guess
- Confetti celebration animation on winning
- Restart game functionality with smooth reset of game and animations
- Responsive and visually appealing design


# How to Play

1. Select a difficulty level.
2. Enter your guess within the specified range.
3. Receive feedback to guide your next guess.
4. Try to guess the correct number within the allowed attempts.
5. Celebrate when you win, or restart and try again!


## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Canvas API for confetti animation
